Demostración de atributos CSS

FLOAT

Este párrafo sirve para mostrar los diferentes atributos de texto aplicados. Tiene margen, relleno, color, fuente, estilo, alineación y sangría.

El texto se ve con fondo, bordes, y una línea de altura ajustada. Además, el elemento flotante a la derecha demuestra cómo el texto se adapta al flujo.

Aquí usamos clear: both; para romper el flujo del elemento flotante anterior y continuar el contenido normalmente.
